Программа создаёт сертификат об участии для образовательных мероприятий.
На вход подаётся PDF в разрешении 150 DPI или PNG-шаблон.
На выходе получаете PNG-изображение на основе шаблона и текстовых полей, которые вставляет программа.
Три текстовых поля с возможностью натройки координат, шрифта и цвета:
- четырёхзначный номер сертификата (можно настроить начальный номер для пакетной обработки)
- имя участника
- дата выдачи

Можно загрузить текстовый список имен (txt-файл с именами в столбик), тогда программа генерирует пачку сертификатов с соответствующими именами и номерами.
